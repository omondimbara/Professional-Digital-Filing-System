# Folder Descriptions

## Documentation

Contains all project documentation developed during the project.

### Reports

Stores the project's formal reports, including:

* Project Proposal
* Project Plan
* Project Report

### Procedures

Contains the Standard Operating Procedures (SOPs) developed for managing digital files.

Examples include:

* Document Filing Procedure
* File Saving Procedure
* File Retrieval Procedure
* Backup and Recovery Procedure
* Archive Management Procedure

### User Guides

Contains instructional documents that explain how to use and maintain the digital filing system.

### References

Contains supporting project documents, including:

* Project Requirements
* Project Objectives
* Project Scope
* Risk Assessment
* Project Timeline
* Resources and Tools
* Quality Assurance Checklist

### Templates

Contains reusable templates developed for future documentation.

Examples include:

* Report Template
* Procedure Template
* User Guide Template
* Reference Document Template
* Checklist Template
